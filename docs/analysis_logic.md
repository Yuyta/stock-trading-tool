# TradeAlgo Pro 判定ロジック詳細

本アプリの分析ロジックは複数の独立したレイヤーで構成されており、選択したトレードスタイル（長期・スイング・デイトレ）に合わせて各レイヤーの重み付けや配点が自動調整されます。

# Layer 0: セクターローテーション分析

市場では常に **資金がセクター間を移動（ローテーション）**しています。  
個別銘柄より先に **どのセクターに資金が流入しているか** を分析することで銘柄選定の精度を向上させます。

## 監視対象
主要セクター指数またはETF
- **米国**: XLK（テック）、XLF（金融）、XLE（エネルギー）、XLY（一般消費）
- **日本**: 銀行、商社、自動車、半導体関連の指数

## 判定ロジック
- **相対強度（Relative Strength）**: 各セクターのパフォーマンスを市場指数（SPY/TOPIX）と比較し、+2%以上のアウトパフォームを「資金流入」、-2%以下を「資金流出」と判定します。

---

## 1. 分析レイヤー概要

### Layer 1: マクロ・リスクフィルター
- **監視指標**:  
  VIX（恐怖指数）、WTI原油、金価格、市場平均（日経225/TOPIX/SPY）。  
  米10年金利（US10Y）、USD/JPY為替、NASDAQ指数。

- **判定ロジック**:
  - **VIX 40超**: 「パニックモード」として全ての新規買い判定をブロック。
  - **VIX 30〜40**: 「高リスク / 警戒モード」。判定は実行されますが、警告ラベルが付与されます。
  - **米10年金利（US10Y） > 4.5%**: グロース株（PER>30）を大幅減点（マクロ補正）。
  - **USD/JPY急騰 (+2%以上)**: 日本株に対して円安メリット加点。
  - **NASDAQが75日線下**: グロース株全体を減点。

---

### Layer 2: ファンダメンタル（中長期）
- **対象**: 日本株（J-Quants / yfinance）、米国株（yfinance）。
- **評価項目**: ROE、営業利益成長率、PER（割安性）、PBR（資産性）。
- **最大点数**: 40点（Growth 20点 / Valuation 20点 [PER:7, PBR:6, ROE:7]）。
- **デイトレモード**: このレイヤーは分析から**完全に除外**されます。

---

### Layer 3: テクニカル（短期〜中長期）
- **指標**:  
  EMA（5/20/75/200日）、RSI（相対力指数）、出来高（RVOL）、MACD、VWAP、ボリンジャーバンド。
- **最大点数**: 40点（スタイルにより重み付け変化）。
- **判定項目**:
  - **長期トレンド**: 200日EMA。長期保有（long_hold）モードにおいて基盤となるトレンド判定に使用。
  - **中期トレンド**: 5/20/75日EMAの位置関係。
  - **出来高 (RVOL)**: 直近20日平均と比較して2倍以上の出来高を「資金流入」として大幅加点。

---

### Layer 4: 定性・ニュース（AI判定）
- **AI**: Gemini AI による感情分析。
- **配点**: スタイルにより 10点（長期）〜 40点（デイトレ）まで変動。
- **ニュース頻度**: 直近24時間のニュース件数が急増している場合に、短期トレンド発生としてボーナス評価。

---

### Layer 5: 配当・インカム（長期保有）
- **対象**: 長期投資家向け。
- **評価項目**: 配当利回り、5年平均配当利回り、配当性向、グレアム指数。
- **最大点数**: 30点。
- **加点ロジック**:
  - **利回り罠の回避**: 異常に高い利回りはリスクとして警告。
  - **配当性向**: 20〜60%を健全とし、80%超は減配リスクとして減点。

---

### Layer 6: 先回り検知・アキュムレーション (Accumulation)
- **目的**: 「これから上がりそう」な蓄積状態を検知するための独立レイヤー。
- **最大点数**: 40点。
- **評価要素 (6項目)**:
  - **ダイバージェンス**: 株価安値付近でRSIが切り上がる逆行現象を検知。
  - **セクター乖離**: Layer 0で特定した強いセクターに対し、個別銘柄が出遅れている状態を評価。
  - **ボラティリティ収縮 (BBスクイーズ)**: ボリンジャーバンドのスクイーズ（幅の縮小）が3日以上継続している状態。
  - **出来高トレンド**: 「静かな蓄積」を評価。急増していない緩やかな増加を検知。
  - **初動トレンド (EMA接近)**: EMA5 と EMA20 が接近しており、反発の初動直前である状態。すでに初動後の場合は除外。
  - **割安成長 (Fundamental連携)**: 低PER（日本株<15, 米国株<20）かつ高成長率を維持している企業を評価（長期・スイング限定、赤字除外）。
- **特殊機能**:
  - **ストッパー**: 直近高値からの急落(-20%)、低流動性、大幅窓開け、出来高減少のいずれかに該当する場合、スコアを無効化（除外）します。
  - **コンボボーナス**: 複数の要素が重なった場合に最大+6点のボーナス加点。
  - **独立スコア**: 「期待値スコア」として算出され、総合判定（Buy/Sell）の重み付け計算には影響を与えない独立指標として扱います。

---

## 2. トレードスタイルによる配点の変化

スタイルに応じて各レイヤーの最大スコアが動的に切り替わります（合計点 / 最大可能点）。

| レイヤー | 長期保有 (配当) | スイング (中長期) | デイトレ (短期) |
| :--- | :---: | :---: | :---: |
| Layer 2: 財務 | 40点 | 40点 | ❌ 除外 |
| Layer 3: テクニカル | 16点 (x0.4) | 32点 (x0.8) | 40点 (x1.0) |
| Layer 4: ニュース | 10点 | 20点 | 40点 |
| Layer 5: 配当 | 30点 | ❌ 除外 | ❌ 除外 |
| **Layer 6: 先回り** | **40点 (独立)** | **40点 (独立)** | **40点 (独立)** |
| **合計最大スコア** | **96点** | **92点** | **80点** |

※ Layer 6 は独立スコアとして表示され、合計最大スコアには合算されません。

---

## 3. シグナル基準 (合計点 / 最大可能点)
各モードや利用可能なAPI（J-Quants/Gemini等）によって最大スコアが動的に変動します。
- **Signal**:
  - 85%以上: `Strong Buy`
  - 65%以上: `Buy`
  - 45%以上: `Hold`
  - それ以外: `Sell / Avoid`

---

## 4. リスク管理

- **流動性チェック**:
  - 銘柄の直近20日平均売買代金を算出。デイトレ等のスタイルに合わせ、一定の閾値（例：日本株5億円）を下回る場合は警告を表示。

- **損切りライン（動的算出）**:
  - **スイング**: 買値から -7% または 60日高値から -10%。
  - **デイトレ**: 買値から -2% または 60日高値から -3%。
  - **長期保有**: 買値から -15%（より広い許容範囲）。

- **時価総額警告**: 日本株100億円未満、米国株1億ドル未満などの小型株に対しボラティリティリスクの警告を付与。

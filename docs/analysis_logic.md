# TradeAlgo Pro 判定ロジック詳細

本アプリの分析ロジックは4つの独立したレイヤーで構成されており、状況に合わせてモードや配点が自動調整されます。

# Layer 0: セクターローテーション分析

市場では常に **資金がセクター間を移動（ローテーション）**しています。  
個別銘柄より先に **どのセクターに資金が流入しているか** を分析することで銘柄選定の精度を向上させます。

## 監視対象
主要セクター指数またはETF
- **米国**: XLK（テック）、XLF（金融）、XLE（エネルギー）、XLY（一般消費）
- **日本**: 1615.T（銀行）、1629.T（商社）、1622.T（自動車）、2644.T（半導体）

## 判定ロジック
- **相対強度（Relative Strength）**: 各セクターのパフォーマンスを市場指数（SPY/TOPIX）と比較し、+2%以上のアウトパフォームを「資金流入」、-2%以下を「資金流出」と判定します。

---

## 1. 分析レイヤー概要

### Layer 1: マクロ・リスクフィルター
- **監視指標**:  
  VIX（恐怖指数）、WTI原油、金価格、市場平均（日経225/TOPIX/SPY）。  
  米10年金利（US10Y）、USD/JPY為替、NASDAQ指数。

- **判定ロジック**:
  - **VIX 40超**: 「パニックモード」として全ての新規買い判定をブロック。
  - **VIX 30〜40**: 「高リスク / 警戒モード」。判定は実行されますが、警告ラベルが付与されます。
  - **NK225/SPY 75日線下**: スイングモード時に「市場注意」ラベルを付与。
  - **米10年金利（US10Y） > 4.5%**: グロース株（PER>30）を減点。
  - **USD/JPY急騰 (+2%以上)**: 日本株に対して円安メリット加点。
  - **NASDAQが75日線下**: グロース株全体を減点。

- **並列処理**: 判定時の遅延を防ぐため、上記指標は `ThreadPoolExecutor` を用いて並列取得されます。

---

### Layer 2: ファンダメンタル（中長期）
- **対象**: 日本株（J-Quants / yfinance）、米国株（yfinance）。
- **評価項目**: ROE、営業利益成長率、PER（割安性）、PBR（資産性）。最大点数は40点（Growth 20点 / Valuation 20点）。

- **加点ロジック**:
  - **PBR < 1 (日本株)**: 東証改革テーマ銘柄として加点（最大6点）。
  - **ROE > 15%**: 高収益企業として加点（最大7点）。
  - **営業利益成長率 > 15%**: 成長株として加点（最大20点）。
    - **異常値保護**: 財務データの不整合（IPO直後や特殊要因）による極端な判定を防ぐため、成長率計算には±500%のキャップ（制限）が適用されます。

- **デイトレモード**: このレイヤーは分析から**完全に除外**されます。

---

### Layer 3: テクニカル（短期〜中長期）
- **指標**:  
  EMA（5/20/75/200日）、RSI（相対力指数）、出来高、MACD、VWAP、ボリンジャーバンド。

- **判定項目**:
  - **EMAトレンド**: 5日>20日 かつ 価格>75日線 で強い上昇トレンドと判定。
  - **ゴールデンクロス**: 5日EMAが20日EMAを上抜いた瞬間に加点。
  - **RSI**: 40〜65を「良好」とし、70超を「過熱」、30未満を「売られすぎ」と判定。
  - **MACD**: MACD線がシグナル線を上抜いている場合にトレンド転換として加点。
  - **出来高 (RVOL)**: 直近20日平均と比較して2倍以上の出来高を「資金流入」として大幅加点。
  - **ボラティリティ**: ボリンジャーバンド（20期間, ±2σ）による過熱・反発期待の判定。
  - **モメンタム**: RVOL（相対出来高）による資金流入判定、VWAP（出来高加重平均）基準での優位性判定。
  - **長期トレンド**: 200日EMA。長期保有（long_hold）モードにおいて基盤となるトレンド判定に使用されます。
  - **チャート可視化**: 5/20/75/200日EMAおよびボリンジャーバンドが価格チャートに直接描写され、視覚的にトレンドとボラティリティを把握可能です。

---

### Layer 4: 定性・ニュース（AI判定）
- **AI**: Gemini AI (`gemini-1.5-flash` 等) による感情分析（0〜20点）。
- **フォールバック判定**: Gemini AI未使用時は、事前に定義されたポジティブ/ネガティブワードの出現頻度に基づきスコアリングします（最大15点）。この際、**具体的にどの単語が検出されたか**が理由欄に表示されます。

- **分析の使い分け**:
  - **スイング**: 中長期的な成長性、不祥事リスク、決算内容の持続性を分析。
  - **デイトレ**: 本日から翌日にかけての「短期的な盛り上がり・材料視」やボラティリティへの影響を最優先。

- **ニュース頻度**: 直近24時間のニュース件数が急増している場合に、短期トレンド発生として最大+4点のボーナス評価。

---

### Layer 5: 配当・インカム（長期保有）
- **対象**: 長期投資家向け。
- **評価項目**: 配当利回り、5年平均配当利回り、配当性向、グレアム指数。最大点数は30点。

- **加点ロジック**:
  - **配当利回り > 3.5%**: 高配当株として加点（最大10点）。
  - **5年平均利回り対比**: 現在の利回りが5年平均を上回っている場合、割安かつ還元強化とみなし加点。
  - **配当性向 20〜60%**: 健全な還元水準として加点。80%超は「減配リスク」として減点。
  - **グレアム指数 (PER × PBR) < 22.5**: 伝統的な割安基準を満たす場合に加点。

## 2. トレードスタイルによる判定の変化

本アプリでは、選択したスタイルに応じて重視するレイヤーと計算ロジックが切り替わります。

| レイヤー | スイング (中長期) | デイトレ (短期) | 長期保有 (配当) |
| :--- | :---: | :---: | :---: |
| Layer 1: マクロ | ✅ 厳格判定 | ✅ 厳格判定 | ⚠️ 参考程度 |
| Layer 2: 財務 | ✅ 重視 (40点) | ❌ 除外 | ✅ 基礎 (20点) |
| Layer 3: テクニカル | ✅ 重視 (40点) | 🔥 最優先 (60点) | ⚠️ 補助 (20点) |
| Layer 4: ニュース | ✅ (20点) | ✅ (40点) | ✅ (10点) |
| Layer 5: 配当 | ❌ 除外 | ❌ 除外 | 🔥 最優先 (30点) |

---

## 3. シグナル基準 (合計点 / 最大可能点)
各モードや利用可能なAPI（J-Quants/Gemini等）によって最大スコアが動的に変動します。
- **Signal**:
  - 85%以上: `Strong Buy`
  - 65%以上: `Buy`
  - 45%以上: `Hold`
  - それ以外: `Sell / Avoid`

---

## 4. リスク管理

- **流動性チェック**:
  - 銘柄の直近20日平均売買代金を算出。デイトレ等のスタイルに合わせ、一定の閾値（例：日本株5億円）を下回る場合は警告を表示。

- **損切りライン（動的算出）**:
  - **スイング**: 買値から -7% または 60日高値から -10%。
  - **デイトレ**: 買値から -2% または 60日高値から -3%。
  - **長期保有**: 買値から -15%（より広い許容範囲）。

- **時価総額警告**: 日本株100億円未満、米国株1億ドル未満などの小型株に対しボラティリティリスクの警告を付与。

# TradeAlgo Pro 利用ガイド

TradeAlgo Pro を最大限に活用するためのセットアップと利用方法についてのガイドです。

## 0. 起動方法

### バックエンドの起動
1. ターミナルで `backend/` ディレクトリに移動します。
2. 仮想環境を有効化します（Windowsの場合: `.\.venv\Scripts\activate`）。
3. サーバーを起動します:
   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload
   ```
   > [!TIP]
   > iPhoneからアクセスする場合は、PCのIPアドレス（例: `http://192.168.1.10:5173`）をブラウザに入力してください。

### フロントエンドの起動
1. プロジェクトのルートディレクトリで新しいターミナルを開きます。
2. 開発用サーバーを起動します:
   ```bash
   npm run dev
   ```
3. 表示されたURL（通常は `http://localhost:5173`）にブラウザでアクセスします。

## 1. 初期設定 (API連携)

本アプリは外部APIと連携することで、高度な分析を提供します。右上の「設定」ボタンから以下のキーを入力してください。

### Google Gemini API キー (推奨)
- **役割**: ニュースの見出しをAIが分析し、投資家の感情（センチメント）を数値化します。
- **設定なしの場合**: キーワードベースの簡易分析（満点7点）に自動で切り替わります。

### J-Quants リフレッシュトークン
- **役割**: 日本株の財務データ（ROE、利益成長率、PER/PBR）を正確に取得します。
- **設定なしの場合**: Yahoo Finance からの分析（fallback）に自動で切り替わります。ただし、より高精度な分析のためには J-Quants の設定を推奨します。

## 2. 基本的な使い方

1.  **銘柄コード / シンボルの入力**
    - 日本株は4桁のコード（例: `7203`）または `.T` 付き（例: `8136.T`）で入力してください。
    - 米国株はティッカーシンボル（例: `AAPL`, `MSFT`）を直接入力します。
2.  **トレードスタイルの選択**
    - **中長期投資 (スイング)**: 日足ベース、ファンダメンタル重視。
    - **短期・デイトレード**: 分足ベース、ボラティリティと短期モメンタム重視（財務分析は除外）。
3.  **時間軸の設定**
    - チャートとテクニカル分析に使用する時間枠を選択します。デイトレなら `5m` や `15m`、スイングなら `1d` が推奨です。
4.  **自動判定を開始**
    - クリックすると4つの分析レイヤーが走り、最終的な売買シグナルが表示されます。

## 3. 分析結果の読み方

- **総合シグナル**: `Strong Buy` / `Buy` / `Hold` / `Sell`
  - 4つのレイヤーの合計スコアを、現在のモードの最大スコアで割った比率で判定します。
- **総合スコア**:
  - 各レイヤーの状況がインジケーターで表示されます。
- **リスク管理**:
  - 現在価格および直近高値からの「損切りライン」が表示されます。トレードスタイルに合わせて、ラインの深さ（-2%〜-10%）が自動調整されます。
- **データソース**:
  - 各項目の末尾に使用されたソース（J-Quants, yfinance, Gemini AI等）が表示され、信頼性を確認できます。

## 4. 分析結果の共有（レポート出力）

「AI 判定結果」カードの右上にある **「共有」ボタン** をクリックすることで、詳細な分析レポートを出力・共有できます。

### 共有方法と動作
1.  **スマートフォンでの利用 (iOS / Android)**:
    - OS標準の共有メニューが開きます。LINE、Slack、メール、メモ帳など、お好きなアプリに結果を送信できます。
    - **画像添付**: 多くの環境では、分析テキストに加えて**現在の価格チャート画像**も自動でキャプチャされ、添付されます。
2.  **PCや非対応ブラウザでの利用**:
    - 共有メニューが利用できない場合、自動的にレポート内容が**クリップボードにコピー**されます。

### レポートの構成
出力されるサマリーには、以下の情報が網羅されています。
- **発行日時**: 分析を実行し、レポートを出力した日時。
- **全レイヤーの情報**: マクロ、ファンダメンタル、テクニカル、定性分析のすべての実数値と分析根拠。
- **リスク管理**: 推奨損切りラインや流動性判定。
- **判定背景**: 各スコアの算出根拠となる詳細な理由。

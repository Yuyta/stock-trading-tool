# TradeAlgo Pro 利用ガイド

TradeAlgo Pro を最大限に活用するためのセットアップと利用方法についてのガイドです。

## 開発・デバッグについて
ローカルでの起動方法や本番環境へのデプロイ手順については、[開発者ガイド](./developer_guide.md)を参照してください。

## 0. 初期設定 (API連携 & ユーザー登録)

本アプリは外部APIと連携し、ユーザーごとの分析履歴を保存することで、高度な分析とパーソナライズされた体験を提供します。

### ユーザー管理 (ログイン・新規登録)
- 右上の「ログイン / 新規登録」ボタンからアカウントを作成できます。
- **ログインのメリット**: 分析結果が自動的にクラウド（データベース）に保存され、後から検索・比較できるようになります。

### Google Gemini API キー (推奨)
- **役割**: ニュースの見出しをAIが分析し、投資家の感情（センチメント）を数値化します。
- **設定**: ハンバーガーメニュー（三本線）の「設定」ボタンから入力してください。
- **設定なしの場合**: キーワードベースの簡易分析（スコアは減少します）に自動で切り替わります。

### J-Quants リフレッシュトークン
- **役割**: 日本株の財務データ（ROE、利益成長率、PER/PBR）を正確に取得します。
- **設定なしの場合**: Yahoo Finance からの分析（fallback）に自動で切り替わります。ただし、より高精度な分析のためには J-Quants の設定を推奨します。

## 1. 基本的な使い方

1.  **銘柄コード / 企業名の入力**
    - **コード直接入力**: 日本株は4桁のコード（例: `7203`）、米国株はティッカーシンボル（例: `AAPL`）を入力します。
    - **企業名検索**: 英語の企業名を入力すると候補が表示されます。**※日本語名での検索は非対応。会社名（英語、例: `Sony`, `Toyota`, `Tomy`）や銘柄コードを入力してください。**
2.  **トレードスタイルの選択**
    - **長期 (配当・インカム)**: 長期保有向け。配当利回り、配当性向、グレアム指数などを重視します。
    - **中長期投資 (スイング)**: 数週〜数ヶ月のスイング向け。日足ベース、ファンダメンタルとテクニカルのバランスを重視。
    - **短期・デイトレード**: 分足ベース、ボラティリティと短期モメンタム重視（財務分析は除外）。
3.  **時間軸の設定**
    - チャートとテクニカル分析に使用する時間枠を選択します。デイトレなら `5m` や `15m`、スイングなら `1d` が推奨です。
4.  **自動判定を開始**
    - クリックすると多層分析レイヤーが走り、最終的な売買シグナルが表示されます。
    - ログインしている場合、判定結果は自動的に履歴に保存されます。

## 2. 履歴の活用

ハンバーガーメニューから「履歴を見る」を選択することで、過去の分析結果にアクセスできます。

- **検索機能**: 銘柄コードで過去の判定を素早く検索。
- **ソート機能**: 分析日時や銘柄名で並び替えが可能。
- **詳細確認**: 履歴リストをクリックすると、当時の分析結果（チャート、各レイヤーのスコア、AIのコメント等）を再現表示できます。

## 3. 分析結果の読み方

- **総合シグナル**: `Strong Buy` / `Buy` / `Hold` / `Sell / Avoid`
  - 各レイヤーの合計スコアを、現在のモードの最大スコアで割った比率で判定します。
  - マクロ環境が極めて不安定（VIX 40超など）な場合は、評価に関わらず「回避推奨」が表示されます。
- **総合スコア**:
  - 各レイヤーの状況がインジケーターで表示されます。
- **リスク管理**:
  - 現在価格および直近高値からの「損切りライン」が表示されます。トレードスタイルに合わせて、ラインの深さ（-2%〜-15%）が自動調整されます。
- **データソース**:
  - 各項目の末尾に使用されたソース（J-Quants, yfinance, Gemini AI等）が表示され、信頼性を確認できます。

## 4. 分析結果の共有（レポート出力）

「AI 判定結果」カードの右上にある **「共有」ボタン** をクリックすることで、詳細な分析レポートを出力・共有できます。

### 共有方法と動作
1.  **スマートフォンでの利用 (iOS / Android)**:
    - OS標準の共有メニューが開きます。LINE、Slack、メール、メモ帳など、お好きなアプリに詳細な分析テキストを送信できます。
2.  **PCや非対応ブラウザでの利用**:
    - 共有メニューが利用できない場合、自動的にレポート内容が**クリップボードにコピー**されます。

### レポートの構成
出力されるサマリーには、以下の情報が網羅されています。
- **発行日時**: 分析を実行し、レポートを出力した日時。
- **全レイヤーの情報**: マクロ、ファンダメンタル、テクニカル、定性、配当インカムのすべての実数値と分析根拠。
- **リスク管理**: 推奨損切りラインや流動性判定。
- **判定背景**: 各スコアの算出根拠となる詳細な理由。

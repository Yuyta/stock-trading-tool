# LLMによるニュース分析×本格チャート指標。AIエージェントAntigravityで実現したモダンなReact+Pythonアプリ構築

## はじめに
近年、AIエージェントによる開発支援が急速に進歩しています。今回は、Google DeepMindが開発した強力なAIエージェント「Antigravity」をペアプログラミング・パートナーとして迎え、本格的な個人投資家向け株分析ツール「TradeAlgo Pro」を構築した体験を共有します。

## TradeAlgo Proとは？
TradeAlgo Proは、単なるチャート表示アプリではありません。「マクロ・セクター・ファンダメンタル・テクニカル・定性」の5層分析を統合し、AIが投資判断をスコアリングするモダンな投資支援プラットフォームです。

### 主な機能
- **5層分析エンジン**:
  - **マクロ/リスク**: VIX、金利、コモディティ価格から市場の地合いを判定。
  - **セクター・ローテーション**: 強いセクターを特定し、トレンドに乗る。
  - **ファンダメンタル**: PER/PBR/ROE、成長率に基づき「割安かつ成長中」の銘柄を抽出。
  - **テクニカル**: EMA、MACD、RSI、ボリンジャーバンド等をチャート上に可視化。
  - **AI定性分析**: Gemini 1.5 Flash（または最新の `gemini-3.1-flash-lite-preview`）を用い、ニュースの真意を読み解く。

## 技術スタック
- **Frontend**: React, TypeScript, Vite, Recharts, Lucide-React
- **Backend**: Python, FastAPI, yfinance, J-Quants API
- **AI**: Gemini API (Google Generative AI)

## Antigravityとの開発体験
Antigravityとの開発で驚かされたのは、その「自律性」と「文脈理解」の深さです。

### 1. 複雑な分析ロジックの構築
「5層の分析結果を統合して100点満点でスコアリングしたい」という抽象的な要望に対し、Antigravityはバックエンドの分析クラス設計から、フロントエンドのスコアバー表示までを一気に提案してくれました。

### 2. モバイルブラウザ特有のデバッグ
iPhoneのSafariで動作が重くなったり、API疎通ができなくなったりする問題に対しても、Antigravityはネットワーク設定（`0.0.0.0` ホスト指定）や、データ取得の並列化による応答速度改善を自ら提案・実行しました。

### 3. 多彩なインジケーターのチャート実装
Rechartsを用いた複雑な多重インデックスチャートの実装において、EMAの計算ロジックから、グラデーションの効いた美しいUIへの統合まで、デザインセンスを含めた実装力を見せつけられました。

## 今回のアップデート：データソースの柔軟性と可視化
当初、日本株の分析にはJ-QuantsのAPIキーが必須でしたが、「yfinanceでも取得可能な情報を fallback として表示したい」というリクエストに対しても、ソースコードを適切にリファクタリングし、どちらのAPIからも最大限の情報を引き出すロジックを実現しました。

また、チャート上に指標（EMA/ボリンジャーバンド）を重ねる際も、視認性を損なわない色使いやダッシュ線の活用など、実用的なアドバイスが光りました。

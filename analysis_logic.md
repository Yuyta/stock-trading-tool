# TradeAlgo Pro 判定ロジック詳細

本アプリの分析ロジックは4つの独立したレイヤーで構成されており、状況に合わせてモードや配点が自動調整されます。

## 1. 分析レイヤー概要

### Layer 1: マクロ・リスクフィルター
- **監視指標**: VIX（恐怖指数）、WTI原油、金価格、市場平均（日経225）。
- **判定ロジック**:
  - **VIX 30超**: 「緊急モード」として全ての新規買い判定をブロック。
  - **VIX 25超**: 「警戒モード」として注意喚起。
  - **NK225 75日線下**: スイングモード時に「市場注意」ラベルを付与。

### Layer 2: ファンダメンタル（中長期）
- **対象**: 日本株（J-Quants / yfinance）、米国株（yfinance）。
- **評価項目**: ROE、営業利益成長率、PER（割安性）、PBR（資産性）。
- **デイトレモード**: このレイヤーは分析から**完全に除外**されます。

### Layer 3: テクニカル（短期〜中長期）
- **指標**: EMA（5/20/75日）、RSI（相対力指数）、出来高。
- **スイング**: 75日線の向き、ゴールデンクロス、RSI（30-60）を重視。
- **デイトレ**: 
  - **5日EMA**との乖離、短期ゴールデンクロスの勢いを重視。
  - **ATR (Average True Range)** によるボラティリティ判定（動かない銘柄は減点）。

### Layer 4: 定性・ニュース（AI判定）
- **AI**: Gemini AI による感情分析（0〜10点）。
- **スイング時プロンプト**: 中長期的な成長への影響を分析。
- **デイトレ時プロンプト**: 
  - 本日から翌日にかけての「短期的な盛り上がり・材料視」を最優先で分析。
  - ボラティリティ（値幅）を伴うニュースかをスコアリング。

## 2. 倍率とノーマライズ（正規化）

各モードによって解析項目が異なるため、合計値を最大スコアで割った「達成率」でシグナルを決定します。

- **シグナル基準**:
  - 85%以上: `Strong Buy`
  - 65%以上: `Buy`
  - 45%以上: `Hold`
  - それ以外: `Sell / Avoid`

## 3. 流動性とリスク管理

- **流動性チェック**:
  - デイトレモード時はスイングの5倍（日本株なら1日5億円以上）の売買代金を要求。不足時は警告を表示。
- **損切りラインの動的調整**:
  - スイング: 買値から -7%、高値から -10%。
  - デイトレ: 買値から -2%、高値から -3%。
- **エラー処理**:
  - 各API（J-Quants, Gemini）が失敗した場合は、自動的にフォールバック（yfinance, キーワード分析等）が行われ、利用可能なデータのみで最大スコアを再計算します。

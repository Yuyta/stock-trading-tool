# TradeAlgo Pro フロントエンド仕様書

## 📄 各ファイルの詳細説明

### 1. `App.tsx`
**役割：メインのUI表示とアプリケーションの状態管理**
* **結果の表示**: バックエンドから返ってきた `AnalysisResult` を以下の要素で可視化します。
  * **総合判定**: シグナル（Strong Buy等）とスコア、分析モードの表示。
  * **マクロバッジ**: VIXや市場トレンドに基づいた安全/警戒/停止のステータス表示。
  * **テクニカルメトリクス**: 5/20/75日EMA、RSI、MACD、VWAP、ボリンジャーバンド、出来高比などの実数値をカード形式で一覧表示。
  * **セクター情報**: 資金が流入している（または流出している）特定のセクターをタグで表示。
  * **リスク管理**: 推奨損切りラインやボラティリティ警告などのリスク情報の表示。
* **チャート機能**: `recharts` を使用。判定結果に応じた色（Buyなら緑、Sellなら赤）がチャートのグラデーションに同期し、直観的な判断を助けます。

### 3. `types.ts`
**役割：TypeScriptの型・インターフェース定義**
* バックエンドの `models.py` と同期された最新の型定義。
* `TechnicalResult` には `macd`, `vwap`, `bollinger_upper/lower`, `ema5/20/75` 等、分析に必要な全ての技術指標が含まれます。
* `MacroResult` には為替トレンド、金利、セクターローテーション等の広範な市場データ構造が定義されています。

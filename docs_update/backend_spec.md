# TradeAlgo Pro バックエンド仕様書

## 📄 各ファイルの詳細説明

### 4. `analyzer.py`
**役割：コア分析アルゴリズムとスコアリング**
* 取得したデータを元に、四層フィルターを通して銘柄の魅力やリスクを算出します。
* 引数・設定により**フルモード**（Gemini/J-Quantsあり）、**標準モード**（どちらかあり）、**基本モード**（APIキーなし）、**デイトレモード** のスコア配分や計算方式をダイナミックに変化させます。
* **1. マクロ環境フィルタ (`_analyze_macro`)**:
  * VIX指数、原油・金価格（シグマ判定）、主要指数（日経/TOPIX/SPY/NASDAQ）を監視。
  * 米10年金利によるグロース株判定、USD/JPYによる為替感度判定を追加。
  * **セクターローテーション分析**: 特定のセクター（半導体、銀行等）への資金流入を市場指数との比較で算出。
* **2. テクニカル分析 (`_analyze_technical`)**:
  * **トレンド**: 5/20/75日EMAの位置関係とゴールデンクロス判定。
  * **オシレーター**: RSIによる過熱感判定。
  * **トレンド転換**: MACDとシグナルのクロス判定。
  * **ボラティリティ**: ボリンジャーバンド（20期間, ±2σ）による過熱・反発期待の判定。
  * **モメンタム**: RVOL（相対出来高）による資金流入判定、VWAP（出来高加重平均）による価格優位性判定。
* **3. ファンダメンタル分析 (`_score_fundamental`)**:
  * PER/PBR/ROEおよび営業利益成長率を分析。日本株特有のPBR1倍割れ改善期待などの加点ロジック。
* **4. 定性・ニュース分析 (`_score_qualitative`)**:
  * Gemini AIによる詳細感情分析。API未設定時はキーワードベースの簡易分析にフォールバック。
  * 直近24時間のニュース件数（出来事の密度）による短期トレンド加点。

# TradeAlgo Pro 利用ガイド

TradeAlgo Pro を最大限に活用するためのセットアップと利用方法についてのガイドです。

## 1. 初期設定 (API連携)

本アプリは外部APIと連携することで、高度な分析を提供します。右上の「設定」ボタンから以下のキーを入力してください。

### Google Gemini API キー (推奨)
- **役割**: ニュースの見出しをAIが分析し、投資家の感情（センチメント）を数値化します。
- **設定なしの場合**: キーワードベースの簡易分析（満点7点）に自動で切り替わります。

### J-Quants リフレッシュトークン
- **役割**: 日本株の財務データ（ROE、利益成長率、PER/PBR）を正確に取得します。
- **設定なしの場合**: Yahoo Finance からの推測データ、または取得不可としてスコアから除外されます。

## 2. 基本的な使い方

1.  **銘柄コード / シンボルの入力**
    - 日本株は4桁のコード（例: `7203`）または `.T` 付き（例: `8136.T`）で入力してください。
    - 米国株はティッカーシンボル（例: `AAPL`, `MSFT`）を直接入力します。
2.  **トレードスタイルの選択**
    - **中長期投資 (スイング)**: 日足ベース、ファンダメンタル重視。
    - **短期・デイトレード**: 分足ベース、ボラティリティと短期モメンタム重視（財務分析は除外）。
3.  **時間軸の設定**
    - チャートとテクニカル分析に使用する時間枠を選択します。デイトレなら `5m` や `15m`、スイングなら `1d` が推奨です。
4.  **自動判定を開始**
    - クリックすると4つの分析レイヤーが走り、最終的な売買シグナルが表示されます。

## 3. 分析結果の読み方

- **総合シグナル**: `Strong Buy` / `Buy` / `Hold` / `Sell`
  - 4つのレイヤーの合計スコアを、現在のモードの最大スコアで割った比率で判定します。
- **総合スコア**:
  - 各レイヤーの状況がインジケーターで表示されます。
- **リスク管理**:
  - 現在価格および直近高値からの「損切りライン」が表示されます。トレードスタイルに合わせて、ラインの深さ（-2%〜-10%）が自動調整されます。
- **データソース**:
  - 各項目の末尾に使用されたソース（J-Quants, yfinance, Gemini AI等）が表示され、信頼性を確認できます。
